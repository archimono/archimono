archimono
=========

**archi**\ medean + ki\ **mono** (monolayer) — a Python toolkit for
systematic structure enumeration of 2D monolayer networks on all 11
Archimedean tilings of the Euclidean plane.

This site is the **API reference**, generated directly from the source
docstrings. For installation and a quickstart, see the project ``README``.
